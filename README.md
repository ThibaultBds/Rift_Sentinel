# Rift Sentinel

Rift Sentinel is a local Python project that analyzes League of Legends match data and gives a simple performance diagnosis.

## Current features

- Calculates average KDA
- Shows most played champions
- Gives a basic diagnosis
- Detects tilt risk
- Gives a short verdict

## Goal

The goal is to build a local analysis engine first, then later turn it into a Discord bot connected to Riot API.

## Roadmap

- Improve diagnosis rules
- Read match data from JSON
- Save analysis history
- Add Discord bot integration
- Connect Riot API later
